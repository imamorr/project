# Currency Converter**Автор:** Халыпенко Егор Олегович**Вариант:** Currency Converter**Дата сдачи:** 30.04.2026---## Описание программы**Currency Converter - это консольное приложение, с помощью которого можно конвертировать валюту по действующему курсу в реальном времени.**---## Требования для запуска1. Перейдите на сайт ExchangeRate-API.com2. Введите свой email3. Нажмите "Get Free Key"4. После входа в панель управления скопируйте ваш API Key5. В файле currency_converter.py замените строку: API_KEY = "ВАШ_API_КЛЮЧ", вставив ваш реальный ключ6. Должна быть установлена библиотека requests---## Как пользоватьсяПри запуске появится меню с пустой строкой, куда нужно ввести сумму.Правее можно выбрать из какой валюты в какую происходит конвертация.История конвертаций сохраняется в файле .json---## Пример работыСумма: 5000 Из валюты: USD В валюту: GBP Конвертировать5000.00 USD = 3706.00 GBPИстория конвертацийДата/время 2026-04-25 16:22:23 Исходная сумма 5000.0 USD Результат 3706.00 GBP Курс 0.7412Можно очистить историю конвертаций---## Как запустить```bashgit clone https://github.com/imamorr/project.gitcd currency_converterpython currency_converter.py
